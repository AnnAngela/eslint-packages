# Contributor Covenant 行为准则

## 我们的承诺

作为成员、贡献者与维护者，我们承诺让每个人都能在社区中无骚扰地参与，不因年龄、体型、可见或不可见的残障、族裔、性征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向而受到区别对待。

我们承诺以开放、友善、多元、包容且健康的方式行动与互动，共同建设社区。

## 我们的标准

有助于营造积极社区氛围的行为包括：

- 对他人保持同理心与善意
- 尊重不同意见、观点与经验
- 乐于给出并坦然接受建设性反馈
- 对自己的错误负责，并向受影响的人致歉，从中吸取经验
- 关注什么对整个社区最有益，而不仅仅是对自己个人最有益

不可接受的行为包括：

- 使用带有性暗示的语言或图像，或任何形式的性骚扰与性关注
- 挑衅、侮辱、贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可公开他人的隐私信息，例如住址或电子邮箱地址
- 其他在专业协作场景下可被合理视为不当的行为

## 执行责任

社区领导者负责解释并执行可接受行为的标准，并将对任何其认为不当、威胁性、冒犯性或有害的行为采取适当且公平的纠正措施。

社区领导者有权移除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、Issue 及其他贡献，并在适当时说明处理原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场合以官方身份代表社区的情况。

例如：使用官方邮箱、使用官方社交媒体账号发言，或以社区指定代表身份参与线上或线下活动。

## 反馈与处理

如遇到辱骂、骚扰或其他不可接受行为，请向负责执行本准则的社区维护者报告：[@AnnAngela](https://github.com/AnnAngela)。

所有投诉都会被及时、公平地审查与处理。

社区维护者有义务保护报告者的隐私与安全。

## 处理分级指南

社区维护者会参考以下“社区影响分级”来决定相应处理方式。

### 1. 更正

**社区影响**：使用了不当语言，或出现其他在社区中不专业、不受欢迎的行为。

**后果**：社区维护者将私下发出书面警告，说明问题性质以及该行为为何不恰当；必要时可能要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为已构成违规。

**后果**：发出正式警告，并说明若继续发生将采取进一步措施；在指定时间内不得与相关人员接触，也不得主动与执行行为准则的人联系。这包括避免在社区空间以及社交媒体等外部渠道进行相关互动。违反此要求可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何公开互动或沟通；在封禁期间，不得公开或私下联系相关人员，也不得主动联系执行行为准则的人。违反此要求可能导致永久封禁。

### 4. 永久封禁

**社区影响**：持续违反社区标准，或对个人进行骚扰，或对特定群体实施攻击、贬损。

**后果**：永久禁止在社区内进行任何形式的公开互动。
